# ``Score``

A Swift framework for building websites using a declarative, type-safe DSL. Score compiles to static HTML, CSS, and minimal JavaScript — no virtual DOM, no runtime framework, no hydration step.

## Overview

Score lets you build server-rendered websites entirely in Swift. Pages, components, and styles are all expressed through Swift code, compiled to optimized static output at build time.

Components that use `@State` emit minimal reactive JavaScript signals for client-side interactivity — everything else is pure static HTML and CSS.
